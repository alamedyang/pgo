Changelog updated 1 Jul 2020

Hello and thanks for stopping by! I've stopped playing Pokemon GO in March 2019, but continued updating the calculator for as long as I could. Sadly, this has come to an end on September 2020.

7/2020
  • added shadow pokemon
  • added buddy boost (L41) option for good to go
  • removed buddy boost levels 42-45
4/2020
  • updated move stats, assigned new moves to some pokemon
12/2019
  • capped CPM level for good to go to 40
  • reworked buddy boost for breakpoint calculator (no longer accounted for in the level 1-40 DPS %, addressed separately in the text info)
  • added CPM levels 41-45 (buddy boost)
  • added new (and old) moves
10/2019
  • reworked attacker/defender drop down (no longer preloads, the list became too long with gen V)
  • enabled fuzzy matching (e.g. if you type in "gradvoir" the search will still manage to return "gardevoir" as one of the results
9/2019
  • added gen V pokemon
  • removed bootstrap (technical debt/performance improvement)
  • updated UI after removing bootstrap
7/2019
  • added temporary friendship bonuses
  • updated content (Armored Mewtwo, new moves for some pokemon
3/2019
  • added changelog
2/2019
  • made pokemon and move pages mobile friendly, added filters
  • added indicators for legacy moves and excluded them from raid boss movesets
  • reworked top counters to be more accurate and flexible, added rank/DPS toggle
  • slightly improved performance and load times
1/2019
  • brought pokemon and move pages back
  • updated moves with PvP data
  • implemented hidden power
3/2018
  • updated the breakpoint calculator for raids
5/2017
  • launched breakpoint calculator beta
