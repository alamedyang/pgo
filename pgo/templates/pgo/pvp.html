The player vs player party planner is a set of tools made to help you create powerful, but balanced pokemon teams.

Start by selecting up to 3 pokemon which will then be placed into slots below the search bar. Pressing each slot will bring the selected pokemon into focus, displaying detailed information about its battling capabilities.

When all slots are full, you can change the content of the one which is currently in focus by searching for another pokemon.
